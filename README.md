# SFTP File Downloader

Ferramenta desenvolvida para automação de download de arquivos utilizando o protocolo SFTP (SSH File Transfer Protocol).

## Descrição
Este projeto tem como objetivo facilitar a transferência de arquivos entre servidores remotos e máquinas locais de forma segura e programática. É uma solução voltada para automação de rotinas de backup, sincronização de dados e integração de sistemas.

## Funcionalidades
* Conexão segura via protocolo SSH.
* Suporte para autenticação por senha ou chave privada.
* Gerenciamento de diretórios locais e remotos.
* Sistema de logs para monitoramento das transferências.

## Tecnologias Utilizadas
* Linguagem: Python
* Principais bibliotecas: paramiko, os, datetime.
